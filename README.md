# AgML Hub

Web interface for browsing [AgML](https://github.com/Project-AgML/AgML) datasets and benchmarks.

## Quick start

```bash
cd frontend
npm install
npm run prebuild   # requires agml-source at ../agml-source
npm run dev
```

See [frontend/README.md](frontend/README.md) for full details.

## Deployment

The frontend deploys to GitHub Pages via Actions. Push to `main` to deploy.

Live at [project-agml.github.io/AgML-Hub](https://project-agml.github.io/AgML-Hub/).
